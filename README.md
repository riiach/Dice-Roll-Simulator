# Dice Roll Simulator

A simple and interactive web-based dice roll simulator that allows users to roll a virtual six-sided die and keep track of their roll history.

## Features

- **Interactive Dice Rolling**: Click the "Roll" button to simulate rolling a six-sided die
- **Visual Dice Display**: Animated dice faces showing the rolled number
- **Roll History**: Track all previous rolls with visual representations
- **History Management**:
  - Expand/collapse history panel
  - Hide/show history
  - Reset history to start fresh
- **Responsive Design**: Works on different screen sizes
- **Attractive UI**: Starry background with modern styling

## Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling and animations
- **JavaScript**: Interactive functionality and dice logic
- **Google Fonts**: Montserrat and Montserrat Alternates font families

## How to Use

1. Open `index.html` in your web browser
2. Click the "Roll" button to roll the dice
3. View the result on the dice face
4. Check the history panel to see all previous rolls
5. Use the control buttons to manage the history:
   - **Expand/Shrink**: Toggle the size of the history panel
   - **Hide/Unhide**: Show or hide the history panel
   - **Reset**: Clear all roll history
